# Python - Server-Side Rendering

This project introduces the basic idea of server-side rendering using Python.

Server-side rendering means generating the final content before it is sent to the user. Instead of writing every page or message manually, we use templates with placeholders and let Python replace those placeholders with real data.

In this project, we create a simple invitation generator. The programme reads a template, takes attendee information from dictionaries, replaces placeholders such as `{name}` and `{event_date}`, and writes personalised invitation files.

The main concepts practised are:

* using string replacement;
* reading from files;
* writing to files;
* working with lists and dictionaries;
* handling missing or invalid data safely.
